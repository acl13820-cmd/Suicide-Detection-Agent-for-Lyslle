This project contains a suicidal-detection chat agent created for Lyslle, for the Practicum class of QMSS in Fall 2025.

The .ipynb file contains the data description as well as documents the model training processes.
The Logistic Regression model and the vectorizer are exported as lr_best and myvec.
The app.py contains the suicide detection agent wrapped, available as a local Streamlit interface. 
The memo file provides detailed documentation of design processes and methodology in the agent creation.
